###Joint　Edit Tool
![](imgs/joint_edit1.png)

####**使い方**
骨を選択し、GUIの操作を行います。

gui            |note
-----------    |-------------------------------
**ShowLabel**  |骨のラベル表示を切り替えます。
**ShowAxis**   |骨の軸表示を切り替えます。
**MirrorJoint**|骨を反転します。付随する属性なども反転します。
